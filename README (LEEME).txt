# SM64-Reloaded-texture-pack
SM64 reloaded texture pack available now for project 1.7 or the plugin for project Jabo'sDirect3d8 1.7.0.57 ver5

Instructions: Create a folder with the name "textures-load" in your Project64 1.7 folder, unzip here the RAR file. In project64 Application, set the Default Video Plugin
to Jabo'sDirect3D8 1.7.0.57-ver5 and do you can start and enjoy the game
